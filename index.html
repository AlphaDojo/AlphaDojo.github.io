Rococo library system

This is a library system that allows its users to track and manage library transactions using an API that is connected to a database. The intention of this system was to increase efficiency since their previous way of handling transactions was done manually without the use of computers.

Since this was a collaborative project, I was tasked with developing the backend system. This included designing the API and the database. I utilized Node.js to develop the API and MYSQL to create the database. I used Sequelize which is a Node.js module to connect the API to the database.

Advenient landing page

This was a group project done during an internship where I learned about front-end design through the lens of marketing and business. The main concept we learned during this time was a marketing funnel and we designed our landing page with that in mind.

This project taught me to be conscientious about the use of page elements to navigate the user seamlessly throughout the site. It also enlightened me on effective front-end design through market funneling.
